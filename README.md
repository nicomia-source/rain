# RAIN

本项目为Unity打包项目，已根据Unity官方.gitignore配置。

## 目录结构
- rain.exe：主程序
- rain_Data/：Unity数据目录
- MonoBleedingEdge/：Mono运行时
- 其它依赖库和资源

## 使用说明
1. 克隆本仓库
2. 运行 rain.exe 启动游戏
